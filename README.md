# Expansión Territorial de Chile

Sitio web educativo sobre la expansión territorial de Chile durante el siglo XIX, centrado en la Ocupación de la Araucanía y la Guerra del Pacífico.

## Descripción

Este proyecto presenta un análisis comparativo entre dos procesos históricos importantes para la formación del territorio chileno: la Ocupación de la Araucanía y la Guerra del Pacífico.

El sitio incluye introducción, desarrollo de cada tema, datos importantes, comparación, conclusión y fuentes consultadas.

## Secciones principales

- Inicio
- Introducción
- Ocupación de la Araucanía
- Guerra del Pacífico
- Datos duros
- Comparación
- Conclusión
- Fuentes

## Tecnologías utilizadas

- HTML5
- CSS3
- GitHub Pages

## Objetivo

Explicar de forma clara cómo estos procesos influyeron en la expansión territorial de Chile, mostrando sus causas, desarrollo y consecuencias.
